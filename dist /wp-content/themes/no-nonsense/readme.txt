=== Introduction ===

Welcome to No Nonsense, the simple but flexible WordPress theme. This theme is meant for those of us who want a simple but well-designed website. No Nonsense supports all WordPress functionality and plays nice with third party plugins as well.


=== Getting Help ===

If you need help with No Nonsese please use the forums on the WordPress themes website:
https://wordpress.org/support/theme/no-nonsense
