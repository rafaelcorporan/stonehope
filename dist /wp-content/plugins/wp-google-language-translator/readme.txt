WP Google Language Translator for WordPress is a free translation plug in that provides instant translations between dozens of different languages.

It can translate words, sentences and web pages between any combination of our supported languages.

It supports more than 60 languages.

== Description ==

The Plug in "WP Google Language Translator" allows you to translate your web page and the whole website in to any language.

Plug in uses more than 60 languages , and this plug in works on Google Language API.

User can also change the language using the front-end language selector box. 

Settings include: Horizontal or Vertical layout. Add the short code [wp-google-language-translator] to pages, posts, and widgets.



== Installation ==

1. Download the zip folder named wp-google-language-translator.zip
2. Unzip the folder and put it in the plugins directory of your wordpress installation. (wp-content/plugins).
3. Activate the plugin through the plug in window in the admin panel.
4. Go to Settings > wp easy translator, enable the plug in, and then choose your settings.
5. Copy the short code [wp-google-language-translator] and paste it into a page, post and widgets.
6. Add widgets in sidebar for posts and pages.
